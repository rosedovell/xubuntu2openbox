### Keyboard Shortcuts

#### Application Launching
```
W-b: Firefox
W-f: Focuswriter
W-p: Pavucontrol
W-t: Terminator
W-e: Thunar
```

#### Desktops
```
C-A-Left/ C-A-Right: Switch to left/right desktop
C-A-S-Left/C-A-S-Right: Move window to left/right desktop + follow
```

#### Windows
```
W-Left / W-Right: Snap windows to left/right sides of screen
W-Up: Maximize
W-q: Close Window
```
